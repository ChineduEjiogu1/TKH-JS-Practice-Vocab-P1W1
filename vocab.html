  • HTTP

    HTTP stands for "Hypertext Transfer Protocol" is a protocol (standard set of rules that allow electronic devices to communicate with each other) or method used to convey information on the web. The original purpose was to provide a way to publish and receive webpages.

  • Client

    A client is a computer hardware device or software that accesses a service made available by a server. Clients are usually seen as the requesting program or user in a client-server architecture.

  • Server

    A server is physical computer or device which purpose is to run services to serve the needs of other computers and its users aka client.

  • Web Browser

    A Web Browser is an application used to access and view websites. The web browser usually render HTML code that make up websites, it process the HTML that includes other resources within a website and the browser processes these assets, then renders them in the browser window.

  • Website

    A website is a collection of many interlinked (connected) or grouped web pages that are accessed by the public that share a single domain name.

  • Web Page

    A Web Page is a document that can be displayed by a web browser and access by entering a URL. A web page may contain text, images, scripts, and hyperlinks to other web pages.

  • JavaScript

    JavaScript is a programming language that primarily used on the web. It is used to enhance websites/webpages. JavaScript is an interpreted language, thus it doesn't need to be complied and is a client-side scripting language, which means the source code is processed by the client's web browser rather than on the web server. JavaScript adds interactivity and dynamically adding and updating content and react to events.

  • HTML

    HTML or Hypertext Markup language is the skelton, framework or structure of a website/webpage. https://www.freecodecamp.org/news/what-is-html-definition-and-meaning/

  • CSS

    CSS is defined as Cascading Style Sheets, it basically the skin, features and aesthetics of webpage/website. It also describes how HTML elements are displayed on screen.

  • URL

    A URL defined as Uniform Resource Locator is a unique identifier to find a resource on the internet. Its commonly know as "web address" due to it locating a web resource like a street address indicates where a person lives. A url URLs consist of multiple parts which includes a protocol, domain name, and resource in the directory structure of the server. that tell a web browser how and where to retrieve a resource.

  • IDE / text editor

    An IDE or Integrated development environment is a software or application for making applications that incorporates customary developer tools into a single graphical user interface (GUI). An IDE consists of Source code editor, local build automation, and debugger. A text editor is a software that is used to create, change, edit and save text files.
